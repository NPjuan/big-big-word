## 💡 使用技巧

### 1. **添加单词**

- 输入单词
- 按 `Enter` 快速提交
- 按 `Esc` 清空输入

### 2. **搜索单词**

- 使用右上角搜索框
- 实时过滤结果

### 3. **查看详情**

- 悬浮查看 Tooltip
- 点击眼睛图标查看详情

### 4. **删除单词**

- 点击删除图标
- 确认对话框

## 🎓 设计亮点

### 1. **Glassmorphism**

```css
background: rgba(255, 255, 255, 0.95)
backdrop-filter: blur(20px)
```

### 2. **渐变系统**

```css
linear-gradient(135deg, color1, color2)
```

### 3. **动画缓动**

```css
cubic-bezier(0.4, 0, 0.2, 1)
```

### 4. **阴影层次**

```css
0 8px 32px rgba(0, 0, 0, 0.1)
```
