## 🎨 设计系统



## 🎬 动画效果

### 1. **页面过渡**

```css
duration: 0.4s
easing: cubic-bezier(0.4, 0, 0.2, 1)
effect: fade + slide
```

### 2. **悬浮效果**

```css
transform: translateY(-4px)
box-shadow: enhanced
transition: 0.3s ease
```

### 3. **脉冲动画**

```css
@keyframes pulse {
  0%, 100%: scale(1)
  50%: scale(1.05)
}
```

### 4. **浮动动画**

```css
@keyframes float {
  0%, 100%: translate(0, 0)
  33%: translate(50px, -50px)
  66%: translate(-50px, 50px)
}
```

### 5. **光泽扫过**

```css
gradient sweep from left to right
duration: 0.5s
trigger: hover
```
