## 🔄 迁移建议

### 从 V4 迁移到 V5

**适合迁移的场景：**

- ✅ 主要用于阅读辅助
- ✅ 需要长时间使用
- ✅ 希望减少视觉干扰
- ✅ 追求专业简洁

**不建议迁移的场景：**

- ❌ 独立学习应用
- ❌ 需要强视觉吸引力
- ❌ 儿童教育产品
- ❌ 短时间使用

### 回滚到 V4

如果V5不适合你的场景：

```bash
# 查看V4的commit ID
git log --oneline | grep "V4"

# 回滚CSS文件
git checkout <v4_commit_id> -- extension/content.css

# 重新加载扩展
chrome://extensions/ > 刷新
```
