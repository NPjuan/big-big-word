# Big Big Word 插件更新说明

## 🎉 最新功能 (v1.1.0)

### 1. 🎯 增强的单词选择功能

#### 选中翻译

- 选中任意单词后，会在单词附近自动显示翻译弹窗
- 弹窗包含音标、词性、详细释义
- 300ms 延迟显示，避免误触

#### 双击翻译

- 双击任意单词即可快速查看翻译
- 自动识别单词边界
- 无需手动选中文本

### 2. 🔊 音标播放功能

- 翻译弹窗中显示音标和播放按钮 🔊
- 点击播放按钮即可听取标准发音
- 播放时按钮会有动画效果
- 自动处理音频加载失败的情况

### 3. 🎴 快速跳转学习页面

#### 翻译弹窗中的跳转

- 在翻译弹窗底部新增"🎴 打开学习"按钮
- 点击后在新标签页打开学习网站
- 网址：https://big-big-word-pevn.vercel.app/

#### 插件弹窗中的跳转

- 插件弹窗底部的"🎴 打开卡片学习"按钮
- 同样跳转到部署的学习网站

### 4. 💾 本地存储与数据同步

- 所有添加的单词保存在 Chrome 本地存储
- 数据格式与网页版完全兼容
- 支持在插件弹窗中查看和管理单词
- 可以清空单词本或删除单个单词

### 5. 🎨 绿色主题设计

- 将原有的紫色主题更新为绿色主题
- 主色调：#10B981 (Emerald-500)
- 渐变色：#059669 (Emerald-600)
- 与网页版保持一致的视觉风格

## 📋 使用方法

### 在网页上使用

1. **选中单词翻译**
   - 用鼠标选中任意英文单词
   - 等待 300ms 后自动显示翻译弹窗
   - 查看音标、词性和释义

2. **双击单词翻译**
   - 双击任意英文单词
   - 立即显示翻译弹窗

3. **播放发音**
   - 在翻译弹窗中点击 🔊 按钮
   - 听取标准英语发音

4. **添加到单词本**
   - 点击"📚 添加到单词本"按钮
   - 单词会保存到本地存储

5. **打开学习页面**
   - 点击"🎴 打开学习"按钮
   - 在新标签页打开学习网站

### 在插件弹窗中使用

1. **查看单词列表**
   - 点击浏览器工具栏的插件图标
   - 查看已保存的所有单词

2. **手动添加单词**
   - 在输入框中输入单词
   - 点击"添加"按钮或按 Enter 键

3. **删除单词**
   - 点击单词卡片右上角的 × 按钮
   - 确认删除

4. **打开学习页面**
   - 点击底部"🎴 打开卡片学习"按钮

5. **清空单词本**
   - 点击底部"🗑️ 清空单词本"按钮
   - 确认清空（不可恢复）

## 🔧 技术细节

### 数据存储格式

```javascript
{
  id: string,              // UUID
  word: string,            // 单词（小写）
  phonetic: string,        // 音标
  audioUrl: string,        // 发音音频 URL
  partOfSpeech: string[],  // 词性数组
  chineseMeaning: [],      // 中文释义
  englishMeaning: [],      // 英文释义
  etymology: {},           // 词源信息
  createdAt: string,       // 创建时间
  reviewCount: number,     // 复习次数
  mastery: number          // 掌握程度
}
```

### 权限说明

- `storage`: 用于保存单词数据
- `activeTab`: 用于在当前标签页注入脚本
- `contextMenus`: 用于右键菜单功能
- `https://api.dictionaryapi.dev/*`: 用于查询单词释义

### API 使用

- 使用 Free Dictionary API 获取单词数据
- API 地址：https://api.dictionaryapi.dev/api/v2/entries/en/{word}
- 免费、无需 API Key
