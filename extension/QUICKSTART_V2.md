# 🚀 快速开始 - Claymorphism 设计版本

## 📦 安装步骤

### 1. 重新加载扩展

如果你已经安装了旧版本：

1. 打开 `chrome://extensions/`
2. 找到 "Big Big Word" 扩展
3. 点击刷新图标 🔄
4. 或者先移除，再重新加载

### 2. 首次安装

1. 打开 `chrome://extensions/`
2. 启用"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `extension` 文件夹

## 🌐 中文翻译功能

### 自动翻译

现在所有英文释义都会自动翻译为中文：

**旧版本（英文）：**

```
noun
• The occurrence of events by chance in a happy way
• A fortunate discovery
```

**新版本（中文）：**

```
noun
• 偶然发生的幸运事件
• 幸运的发现
```

### 翻译 API

- **服务**：LibreTranslate（免费、开源）
- **方向**：英文 → 中文
- **内容**：单词释义
- **速度**：通常 < 1 秒

---

## 🎯 使用方法

### 1. 选中翻译

1. 在网页上选中任意英文单词
2. 等待 300ms
3. 弹窗自动出现
4. 查看中文释义

### 2. 双击翻译

1. 双击任意英文单词
2. 立即显示翻译弹窗
3. 无需手动选中

### 3. 播放发音

1. 点击音标旁的 🔊 按钮
2. 听取标准英语发音
3. 播放时按钮有脉冲动画

### 4. 添加单词

1. 点击"📚 添加到单词本"
2. 单词保存到本地
3. 在插件弹窗中查看

### 5. 打开学习

1. 点击"🎴 打开学习"
2. 跳转到学习网站
3. 开始卡片复习

## 🎨 设计细节

### 配色方案

```css
/* 主色调 */
Primary:    #0D9488  /* 深青色 */
Secondary:  #2DD4BF  /* 亮青色 */
CTA:        #EA580C  /* 橙色 */
Background: #F0FDFA  /* 浅青背景 */
```

### 字体系统

- **标题**：Baloo 2（playful, friendly）
- **音标**：SF Mono / Monaco（等宽）

### 边框厚度

- **弹窗**：3px
- **卡片**：3px
- **按钮**：2-3px
- **音标区**：2px

## ⚡ 性能优化

### 翻译缓存

- 已翻译的释义会缓存
- 避免重复调用 API
- 提升响应速度

### 动画性能

- 使用 `transform` 和 `opacity`
- 避免 `width` 和 `height` 动画
- GPU 加速

### 懒加载

- 字体按需加载
- 音频按需播放
- 减少初始加载时间

## 🐛 故障排除

### 翻译失败

**问题**：释义显示为英文

**解决**：

1. 检查网络连接
2. 确认 LibreTranslate API 可访问
3. 查看浏览器控制台错误信息

### 样式异常

**问题**：弹窗样式不正确

**解决**：

1. 清除浏览器缓存
2. 重新加载扩展
3. 确认 `content.css` 已更新

### 音频无法播放

**问题**：点击 🔊 无反应

**解决**：

1. 检查浏览器音量
2. 确认网站允许音频播放
3. 部分单词可能没有音频
