# 🚀 Big Big Word Extension - 快速开始指南

## 📦 5分钟快速上手

### 第一步：安装插件

1. **打开 Chrome 扩展管理页面**

   ```
   在地址栏输入：chrome://extensions/
   ```

2. **启用开发者模式**
   - 点击右上角的"开发者模式"开关

3. **加载插件**
   - 点击"加载已解压的扩展程序"
   - 选择 `big-big-word/extension` 文件夹
   - 点击"选择文件夹"

4. **固定到工具栏**（可选）
   - 点击浏览器工具栏的拼图图标
   - 找到 "Big Big Word"
   - 点击图钉图标固定

### 第二步：测试功能

1. **打开测试页面**

   ```
   在浏览器中打开：extension/test.html
   ```

2. **测试选中翻译**
   - 用鼠标选中任意单词
   - 等待翻译弹窗出现
   - 查看音标和释义

3. **测试双击翻译**
   - 双击任意单词
   - 立即显示翻译弹窗

4. **测试音频播放**
   - 选中单词后，点击弹窗中的 🔊 按钮
   - 听取标准发音

5. **测试保存单词**
   - 点击"📚 添加到单词本"按钮
   - 打开插件弹窗查看已保存的单词

6. **测试快速跳转**
   - 点击"🎴 打开学习"按钮
   - 跳转到学习网站

### 第三步：在实际网页中使用

1. **访问任意英文网页**
   - 例如：Wikipedia、Medium、BBC News 等

2. **开始学习单词**
   - 遇到不认识的单词，选中或双击
   - 查看翻译和音标
   - 播放发音
   - 添加到单词本

3. **管理单词本**
   - 点击浏览器工具栏的插件图标
   - 查看已保存的单词列表
   - 删除不需要的单词
   - 或清空整个单词本

4. **打开学习应用**
   - 在插件弹窗中点击"🎴 打开卡片学习"
   - 在学习网站中复习单词

## ❓ 常见问题

### Q1: 选中单词后没有显示翻译？

**A:**

- 确保插件已启用
- 刷新网页重新加载
- 检查是否选中了单个单词（不支持多个单词）

### Q2: 无法播放音频？

**A:**

- 部分单词可能没有音频
- 检查浏览器音量设置
- 某些网站可能阻止音频播放

### Q3: 添加单词失败？

**A:**

- 检查网络连接
- 确认单词拼写正确
- 单词可能已存在于单词本中

### Q4: 如何导出单词？

**A:**

- 目前需要手动导出
- 打开浏览器控制台
- 运行：`chrome.storage.local.get(['big-big-words'], console.log)`
- 复制输出的 JSON 数据

### Q5: 如何同步到其他设备？

**A:**

- 目前不支持自动同步
- 可以手动导出/导入数据
- 未来版本会支持云端同步

## 🔧 故障排除

### 问题：插件无法加载

**解决方案：**

1. 确认已启用开发者模式
2. 检查文件夹路径是否正确
3. 查看是否有错误提示
4. 尝试重新加载插件

### 问题：翻译弹窗位置不对

**解决方案：**

1. 关闭弹窗重新打开
2. 刷新网页
3. 调整浏览器缩放比例

### 问题：数据丢失

**解决方案：**

1. 检查是否误删了单词
2. 确认没有清空单词本
3. 卸载插件会清除数据，需要提前备份
