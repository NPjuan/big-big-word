# V5.1 快速测试指南

## 🚀 快速开始

### 1. 重新加载扩展

```bash
# 打开 Chrome 扩展管理页面
chrome://extensions/

# 找到 "Big Big Word"
# 点击刷新按钮 🔄
```

### 2. 打开测试页面

```bash
# 方式 1: 使用内置测试页面
open extension/test.html

# 方式 2: 打开任意英文网页
# 例如：https://en.wikipedia.org/wiki/Serendipity
```

### 3. 测试功能

#### 测试 1: 双击单词

```
1. 双击单词 "serendipity"
2. 观察加载过程：
   - 显示 "查询中..." (0.5-1秒)
   - 显示 "正在翻译..." (1-2秒)
   - 显示中文翻译（仅中文，无英文）
```

#### 测试 2: 验证纯中文

```
✅ 应该看到：
- 词性标签（如：noun）
- 中文释义列表
- 音标和播放按钮

❌ 不应该看到：
- 英文释义
- 英文例句
```

#### 测试 3: 音标播放

```
1. 点击播放按钮 🔊
2. 应该听到单词发音
3. 按钮应该有播放状态反馈
```

## 🐛 常见问题

### 问题 1: 弹窗不显示

**原因:** 扩展未重新加载

**解决:**

```bash
chrome://extensions/ > 刷新 Big Big Word
```

### 问题 2: 显示英文释义

**原因:** 使用了旧版本代码

**解决:**

```bash
# 确认使用最新代码
git pull origin main

# 重新加载扩展
chrome://extensions/ > 刷新
```

### 问题 3: 翻译失败

**原因:** LibreTranslate API 不可用

**解决:**

```bash
# 检查网络连接
# 检查 API 是否可访问
curl https://libretranslate.com/translate

# 如果 API 不可用，等待恢复
```

### 问题 4: 音频无法播放

**原因:** 音频 URL 无效或网络问题

**解决:**

```bash
# 检查控制台错误信息
# 尝试其他单词
# 检查网络连接
```
